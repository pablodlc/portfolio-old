About Me

Hello World! My name is Pablo and I'm a fledgling programmer. I have a varied background; food service, retail, social work, law, and currently I'm a technical writer for a scientific instrument manufacturing company. I made a cookie cutter webpage long ago and thought it was great fun. Somehow it never occurred to me that I could enjoy it professionally. I'm enthusiastically learning new technologies and concepts. I definitely consider myself a full-stack programmer, as I equally enjoy making a site look good and do cool things with CSS and other styling tools, and backend work, such as communicating with server-side APIs to fetch pertinent data.


At this point in my learning process, I can build an interactive site from scratch, that can record user input or communicate with a server-side API, generate a password, or make a JavaScript powered game. I love JavaScript and how powerful it is in making a page fun and dynamic. jQuery is a handy shorthand I'm familiarizing myself with and using often. I can use CDNs to customize fonts and include sharp looking icons, I can fetch data from server-side APIs and return it to a page. I'm also gaining experience in JavaScript libraries, like Moment.js, and CSS frameworks; specifically Bootstrap and Bulma.

Trivia with a Twist!

This is a group project assignment where two classmates and I made a drinking trivia application. We were to use at least two server-side APIs, local storage, and have it be interactive. We made this good looking, fun application that generates random drinks and runs through a 5 question general knowledge trivia game, each using a different API.

Password Generator

A password generator where the user can generate a password with a length of 8 to 128 characters, and can choose to include or exclude lowercase letters, uppercase letters, numbers, and special characters.

Work Day Scheduler

This is a scheduler application in the style of a daily planner. It's persistent, utilizing local storage to save user input. Work Day Scheduler also uses Moment.js to dynamically update elements on the page. For best results, view this page between the hours of 10:00 A.M. and 4 P.M. That allows you to see all three background colors that dynamically update.

Taskmaster Pro Edition

An assignment similar to Work Day Scheduler. This application taught me about jQuery and utilizing Moment.js, as well as some more advanced CSS styling.

Run Buddy

Run Buddy's landing page, made with HTML and CSS, to include Flexbox and media queries.

Horiseon

A code refactor assignment to improve Horiseon's homepage's accessibility. This lesson taught me the importance of accessability in websites, and how simple elements like semantic HTML and alt tags in images greatly improve the user experience for people using screen readers.

RAMONES Fanpage

A vanilla HTML fan page for the greatest band of all time. Ever. This was a pre-work assignment for my boot camp to familiarize us with basic HTML. I include it because, mostly, it's the Ramones. But also to demonstrate my earliest efforts.
